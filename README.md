# MyPlayer
React-Native

# Installation<br>
<b> For Android run : </b><br>
1- yarn install<br>
2- yarn android<br><br>
<b> For IOS </b><br>
1- yarn install<br>
2- cd ios <br>
3- pod install <br>
4- cd ../ <br>
5- yarn ios<br>

# Technologies<br>
Native Base, Reanimated 2, Context, Async storage, Navigation, Flatlist, google signin, Gesture Recognizer, ... .<br>
